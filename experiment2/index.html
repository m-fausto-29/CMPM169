What code did you start with? (You don't need to provide the code itself, but where did you start?) Document this step briefly.

The code that I mainly started with for this experiment include p5.js code generated from Wesbot, an example about how to generate smoke particles from the p5.js website, and code that I found by Katie Liu who created a sketch that generated multiple flowers.

Integrate

After learning from existing techniques and strategies, we can build upon that foundation by combining elements from different sources. This might involve combining elements from different programming languages, using libraries or frameworks in new ways, or bringing together techniques from different fields or disciplines. By integrating and combining elements from different sources, we can create something new and unique.

How did you extend the code or combine with another technique to make something new? Document this step.

For the gradient effect, I altered the code so that I would get the right color scale that I desired and made sure that the effect went from dark to light rather than light to dark. For the particle effect, instead of using the example as a base, I merely used it as inspiration for my own system that's a lot simpler, yet was able to achieve the effect that I wanted. For the flowers, I used the code that I found from Katie Liu by changing the amount of flowers generated, along with the shape and visual appearance.

Innovate

Finally, we can introduce new or original ideas to our coding process by thinking creatively and pushing the boundaries. How did you use the code as a jumping off point to create something truly unique and novel? Document this step thoroughly.

After getting familiar with the technqiues I learned, I used that knowledge to enhance some of the pieces that I had already, for example, the ground and the flowers. I applied the gradient effect to the ground to give it more character and included the mouseclicked() function to make it so the user can make the flowers appear and change their colors each time they clicked on the canvas.
